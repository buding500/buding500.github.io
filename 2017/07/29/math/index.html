数字相关Math对象和Number对象常用方法

发表于 2017-07-29 | 分类于 javascript

在项目开发过程中，经常用到一些数字的计算与转化，觉得还是总结一下比较好。跟数字相关的javascript对象有Math和Number,下面是一些常用方法：

Math对象没有构建函数Math(),使用时无需先new一个，直接使用方法即可。

常用的Math对象方法：
abs(x)：返回x绝对值
min(x,y)：返回x、y中最小值
max(x,y)：返回x、y中最大值
ceil(x)：对x进行向上舍入成整数
floor(x)：对x进行向下舍入成整数
round(x)：对x进行四舍五入成整数
random()：返回 0 ~ 1 之间的随机数
Number对象有构建函数，它最有用的方法是toFixed():
toFixed(n): 将number四舍五入为指定小数位数的数字。参数必须，为小数位数，值为0-20

foFixed()用来截取小数位非常好用，不足的可以补0。
注意几点：

1、n必须是Number类型
2、返回的是string类型
